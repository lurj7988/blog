                  # Installation and Running

                  Make sure the required dependenciesopen in new window are met and follow the instructions available for both NVidiaopen in new window (recommended) and AMDopen in new window GPUs.

                  Alternatively, use online services (like Google Colab):

                  • List of Online Servicesopen in new window

                  # Automatic Installation on Windows

                  1. Install Python 3.10.6open in new window, checking "Add Python to PATH"
                  2. Install gitopen in new window.
                  3. Download the stable-diffusion-webui repository, for example by running git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git.
                  4. Place stable diffusion checkpoint (model.ckpt) in the models/Stable-diffusion directory (see dependenciesopen in new window for where to get it).
                  5. Run webui-user.bat from Windows Explorer as normal, non-administrator, user.

                  # Automatic Installation on Linux

                  1. Install the dependencies:
                  # Debian-based:
sudo apt install wget git python3 python3-venv
# Red Hat-based:
sudo dnf install wget git python3
# Arch-based:
sudo pacman -S wget git python3
                  1. To install in /home/$(whoami)/stable-diffusion-webui/, run:
                  bash <(wget -qO- https://raw.githubusercontent.com/AUTOMATIC1111/stable-diffusion-webui/master/webui.sh)
